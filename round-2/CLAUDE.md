# CLAUDE.md

## Project Rules

### Rule 1
Always validate user input before form submission.

### Rule 2
Use semantic HTML elements and properly associated labels.

### Rule 3
Keep JavaScript modular and avoid inline event handlers.

### Rule 4
Display inline validation messages instead of browser alerts.

### Rule 5
Review all AI-generated code before committing changes.

### Rule 6
Separate structure (HTML), styling (CSS), and behavior (JavaScript).

### Rule 7
Use descriptive IDs and class names for maintainability.
