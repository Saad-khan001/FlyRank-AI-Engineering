# Notes

This document contains prompts, observations, and key learnings recorded during the completion of **FE-03 – The AI-Assisted Workflow Drill**.

---

# Prompts

## Round 1 (Vague Prompt)

```text
Create a login and signup page.
```

The objective was to provide a minimal prompt without additional context, planning, constraints, or verification steps.

---

## Round 2 (Structured Prompt)

The implementation followed a structured AI-assisted workflow.

The AI was instructed to:

- Review the existing project structure.
- Create a development plan before writing code.
- Build a responsive Login & Signup Authentication Form.
- Use semantic HTML.
- Apply responsive CSS.
- Implement client-side validation using Vanilla JavaScript.
- Improve accessibility.
- Handle common edge cases.
- Verify the implementation before completion.
- Review and fix AI-generated issues before finalizing the solution.

---

# Observations

## Round 1

- The AI generated a working authentication interface quickly.
- The prompt lacked context and technical requirements.
- The generated implementation required additional review and improvements.
- Some functionality and validation logic were incomplete.

## Round 2

- The AI first created a development plan before implementation.
- The workflow was organized into HTML, CSS, and JavaScript phases.
- Validation, accessibility, and responsive design were included.
- The AI performed a review step and identified issues before completion.
- Minor integration issues, such as the missing JavaScript file reference, were detected and corrected during verification.

---

# Learnings

- Clear prompts produce higher-quality AI-generated code.
- Planning before coding reduces development time during review.
- AI-generated code should always be verified before submission.
- Git branches make it easier to compare different implementation approaches.
- Accessibility and validation should be considered from the beginning of development.
- A structured AI workflow produces cleaner, more maintainable code than relying on a single vague prompt.
